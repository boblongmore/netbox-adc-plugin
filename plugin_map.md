### Plugin Map

```
+-- Nodes
    +-- Node Name
        +-- Node properties
            +-- description
            +-- address or FQDN

+-- Pools
    +-- Pool Name
        +-- Description
        +-- Pool Members
        +-- Pool Properties
            +-- load balancing method

+-- Virtual servers
    +-- device/cluster
    +-- Name
        +-- description
        +-- type
        +-- Source address
        +-- Destination address/mask
        +-- Port
        +-- state
        +-- protocol
        +-- default pool
```
